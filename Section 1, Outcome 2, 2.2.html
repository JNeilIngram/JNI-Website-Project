2.2 Describe whether the IT tools selected were appropriate E.g. Visual Studio Code, Codepen, Trello

Trello had some use, but as the project is a solo one, it actually detracted from efficiency rather than contributed to it. This is because it became its own source of work rather than being a guide to progress. If the project was a team one it would have been more relevant and useful.

Visual Studio Code was the web document creation tool. This is quite user friendly if you know what you are doing. Indeed if the tuition had been geared more like the earlier python tuition, with multiple small example tasks to build skills before starting on a full project, the whole exercise would have been easier.

The Website W3schools.com was particularly helpful when the tuition was lacking or absent (such as creation of anavigation bar).
