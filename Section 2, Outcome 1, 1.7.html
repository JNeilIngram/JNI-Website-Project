1.7 Describe which are the appropriate filetypes for websites

Various file types are used to produce any website. The precise mix of files will be determined by the nature of the website. A list of some of the more common types (with brief desciptions) appears below [3]:-

Image files:-

.JPG/.JPEG
common image file - often used for photographs
.PNG
common image file - tends to be larger than .JPG/.JPEG
.TIFF
image file for very high quality images - may not be compatible with most browsers

Video files:-

.AVI
non-streamable video file
.GIF
very short video clip
.MPG/.MPEG
streamable video file
.MOV/.QT
non-streamable video file, initially geared towards Apple Macs, but now in common usage
.RAM
non-streamable video and audio file format currently out of favour

Audio files:-

.MID/.MIDI
music files
.MP3
audio/music files - highly compressed
.WAV
audio files - limited compression if any

Archives:-

.RAR
standard archive file
.ZIP
standard archive file

Text and formatting files:-

.CSS
Web page design document with extensive formatting options
.DOC/.DOCX
Microsoft Word document
.HTML/.HTM
Web page design document with limited formatting options
.PDF
Adobe Acrobat document - not easily editable without the Adobe Acrobat os similar PDF manipulation software, although readers are free
.TXT
the simplest of all text files containing just text

Other files:-

.EXE
EXEcutable file - typically used in installers
Streaming Media
various audio and video files that are capable of playing directly from the webite (as opposed to being downloaded and played locally)
